Observable W3C Web Components
===========================


# Installation

##npm

``` bash

git clone https://github.com/ellipticaljs/component.git
npm install

```

##bower

``` bash

bower install elliptical-component

```

##demo

``` bash

git clone https://github.com/ellipticaljs/component.git
npm install
bower install
gulp demo
python -m SimpleHTTPServer 8080


```

browser: localhost:8080
